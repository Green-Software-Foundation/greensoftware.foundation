## 3. How to Publish Articles / Stories / Research

**All three follow the same pattern:**

1. Go to `https://greensoftware.foundation/admin/`
2. Log in with GitHub (you need repo write access)
3. Pick the collection (Articles, Stories, or Research)
4. Click "New" — fill in the fields
5. It saves with `published: false` by default — the page exists at its URL but is hidden from listings, search, and sitemap
6. Share the direct URL for review
7. When approved, edit and set `published: true`

**Articles** support: co-located images, featured flag, tags, authors with photos/social links, translations (EN/JA/PT/ZH)

**Research** supports: DOI, PDF link, status (draft/in-progress/published), jurisdiction, framework, author affiliations

**Stories** support: rich narrative fields — timeline, stats, contributor profiles, featured quotes, CTAs

**Rich content in articles:** Authors can use special markdown syntax for YouTube embeds (`::youtube{id="..."}`), Google Slides, callout boxes, buttons, and cross-links to other articles. There's a showcase at `/articles/rich-content-showcase/` with every example.
